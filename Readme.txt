# คำอธิบายของ Source Code แต่ละ Task
1. Classification (การทำนายกลุ่มผู้ก่อการร้าย)
    1.1 Import Python Library ที่จำเป็นต้องใช้งาน ได้แก่ Matplotlib, Pandas, Shapely, Seaborn, GeoPandas, Numpy, scikit-learn เป็นต้น
    1.2 อ่านไฟล์ CSV โดยใช้ Pandas
    1.3 นำข้อมูลจุดเกิดเหตุ (ลองติจูด ละติจูด) มาแสดงบนแผนที่โลกและสร้างกราฟระบุสีจุดตามกลุ่มของผู้ก่อการร้าย
    1.4 เลือก Feature และ Label สำหรับนำไปฝึกฝนและการตรวจสอบถูกต้องความแม่นยำของแบบจำลอง
    1.5 ตรวจสอบและนับจำนวนข้อมูลที่มีค่า Null/NaN
    1.6 แทนค่า Null/NaN โดยใช้ค่า Mean ของแต่ละคอลัมน์ และนับจำนวนข้อมูลที่มีค่า Null/NaN อีกครั้ง
    1.7 แยกข้อมูลออกเป็น Feature (X) และ Label (y)
    1.8 นับจำนวนความหลากหลายของข้อมูลในแต่ละคอลัมน์เพื่อนำมาพิจารณาตัด Feature บางส่วนออก
    1.9 ใช้ Principal Component Analysis (PCA) มาตรวจสอบมิติของข้อมูลต่าง ๆ เปรียบเทียบกับการอธิบายของข้อมูลที่เพิ่มมากขึ้น
    1.10 การปรับช่วงข้อมูล (Feature Scaling) หรือ Data Normalization ในรูปแบบต่าง ๆ
    1.11 สุ่มข้อมูลและแบ่งข้อมูลเพื่อนำไป Train 70 % และ Validation 30 % โดยใช้ Train Test Split
    1.12 การฝึกฝนแบบจำลองโดยใช้ Algorithm ที่หลากหลายรูปแบบ และทำการประเมินผลลัพธ์เพื่อการตรวจสอบความถูกต้องและความแม่นยำของแบบจำลอง
    1.13 ทดสอบแบบจำลองกับข้อมูล Test Dataset และนำผลลัพธ์จากการทำนายอัพโหลดขึ้นระบบ
2. Regression (การทำนายจำนวนผู้ติดเชื่อ COVID-19)
    2.1 Import Library ที่จำเป็นต้องใช้งาน ได้แก่ Pandas, Numpy, Plotly Express, Scikit-learn และ Prophet
    2.2 นำเข้าข้อมูล COVID-19 จาก github โดยใช้ Pandas ในการอ่านไฟล์ CSV
    2.3 จัดการกับค่า null/NaN กับข้อมูลประเภท float64 ด้วยการแทน 0
    2.4 filter ข้อมูลให้เหลือเฉพาะข้อมูลก่อนวันที่ 17 เมษายน 2022 เนื่องจาก task นี้ต้องการที่จะทำนายจำนวนผู้ติดเชื้อโดยรวมล่วงหน้า 7 วัน ได้แก่วันที่ 17-23 เมษายน 2022
    2.5 filter ข้อมูลให้เหลือเฉพาะข้อมูลของพื้นที่ที่ต้องการจะทำนาย ได้แก่ World, Indonesia, Vietnam, Thailand, Philippines, Singapore, Malaysia, Myanmar, Laos, Cambodia, Brunei และ Timor-Leste
    2.6 สร้างแผนภาพจาก Bar chart โดยแสดงการเพิ่มขึ้นของจำนวนผู้ติดเชื้อโดยรวม และจำนวนผู้ติดเชื้อรายวัน ทั้งโลกและแต่ละประเทศ เพื่อวิเคราะห์ข้อมูลเชิงลึก
    2.7 สร้างฟังก์ชันที่สร้างแบบจำลองโดยใช้อัลกอริทึม Linear Regression และ Support Vector Machine โดยมีการแบ่งข้อมูล train 70% สำหรับฝึกฝนแบบจำลอง และ test set 30% สำหรับวัดประสิทธิภาพแบบจำลอง
    2.8 สร้างฟังก์ชันที่สร้างแบบจำลองและวัดประสิทธิภาพโดยใช้อัลกอริทึม Prophet โดยไม่มีการแบ่งข้อมูล
    2.9 เรียกใช้ฟังก์ชันที่สร้างไว้สำหรับการสร้างแบบจำลองแต่ละอัลกอริทึม โดยข้อมูลส่งเข้าฟังก์ชันจะเป็นข้อมูลตามพื้นที่ เพื่อทำนายจำนวนผู้ติดเชื้อโดยรวมล่วงหน้า 7 วัน ในแต่ละพื้นที่ที่ได้กล่าวใน 2.5
    2.10 เปรียบเทียบประสิทธิภาพแบบจำลองด้วยค่า MSE และเลือกแบบจำลองที่ ดีที่สุดจากแบบจำลองที่มีค่า MSE โดยเฉลี่ยจากการทำนายทุกพื้นที่ต่ำที่สุด

    3.1 Import Python Library ที่ใช้งาน เช่น Pandas, Seaborn, KMeans, scikit-learn, metrics เป็นต้น

    3.3 สำรวจข้อมูลและสถิติพื้นฐานของข้อมูล
    3.3 ตรวจสอบจำนวนค่าว่าง หรือ Null value ในข้อมูล
